## ⚙️ Features

<table>
  <tr>
    <td width="50%">
      <h3>📤 Dual Input Options</h3>
      Upload PDF/DOCX files or paste resume text directly with a simple toggle interface
    </td>
    <td width="50%">
      <h3>🤖 AI-Powered Analysis</h3>
      Get comprehensive feedback powered by Groq's Llama 3.3 70B model
    </td>
  </tr>
  <tr>
    <td>
      <h3>📊 Overall Scoring</h3>
      Receive a score out of 100 with animated display and summary feedback
    </td>
    <td>
      <h3>🎯 Section-by-Section Feedback</h3>
      Detailed suggestions for Header, Summary, Experience, Projects, Skills, and Education
    </td>
  </tr>
  <tr>
    <td>
      <h3>✨ Bullet Improvements</h3>
      Specific rewritten bullet points to improve impact with action verbs and metrics
    </td>
    <td>
      <h3>🔑 ATS Optimization</h3>
      Missing keywords identification for better applicant tracking system matching
    </td>
  </tr>
  <tr>
    <td>
      <h3>🎨 Modern UI</h3>
      Clean, minimal design inspired by Notion/Linear/Stripe with soft shadows and rounded cards
    </td>
    <td>
      <h3>📱 Responsive Design</h3>
      Works seamlessly on desktop and mobile devices
    </td>
  </tr>
</table>

## 🔐 Privacy First

> **Your resume data stays secure.** API keys are only used for your current session and never stored. All file processing happens locally in your browser before sending text to the AI API.
